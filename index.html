MoveMate (SFHacks 2025)

Inclusive Fitness App for the Visually Impaired

MoveMate is a web-based, voice-guided fitness app designed for visually impaired users. Most workout platforms rely on visuals, leaving out a key community. MoveMate solves this by combining voice commands, audio instructions, and real-time body tracking to guide users through basic exercises, all screen-free and hands-free.
Hackathon project built in 2 days at SF Hacks 2025 by a team of 4.

 This project features: • Real-time body tracking with TensorFlow.js and MoveNet • Voice input and feedback using the Web Speech API • Fully audio accessible interface with high contrast and large fonts • HTML, CSS, and JavaScript-based frontend • Designed for independence, safety, and confidence • Built with inclusive UX principles and browser compatibility in mind
